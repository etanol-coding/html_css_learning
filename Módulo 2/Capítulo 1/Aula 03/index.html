Harmonia de Cores

Na hora de montar o seu site, você precisa escolher uma paleta de cores, e não adianta sair testando diversas paletas até chegar a um resultado "bonito". Os designers utilizam cores harmonicas para fazer isso ficar bonito.

Cores para escrever

Sempre que for escrever no seu site, é recomendável que use cores de alto contraste, por isso sempre que realmente for escrever um texto minimamente grande use o fundo branco e a cor preta.

Cansaço Visual

Se você usar o fundo preto e a cor branca pode causar cansaço visual, o mesmo para quando for escrever em uma cor que não seja preto e branco.

Muito texto Colorido

Sempre que for escrever em colorido ou com o fundo preto, você não pode colocar muito texto, quando for realmente colocar texto de verdade use o fundo branco e a cor preta.

Onde usar textos coloridos?

Use textos coloridos somente para textos curtos como textos de boas vindas: "Olá! Bem vindo!" ou mensagens bem curtas como slogans.
