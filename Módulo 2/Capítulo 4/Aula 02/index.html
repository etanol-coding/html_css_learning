Decoração das Bordas

Nas CSS você pode configurar o estilo de uma borda usando border-style: ;, esse comando pode receber:

  • border-style: solid;
  • border-style: dashed;
  • border-style: dotted;
  • border-style: double;
  • border-style: groove;

IMPORTANTE

Nas CSS os termos para cima baixo esquerda e direita são respectivamente: top bottom left right
