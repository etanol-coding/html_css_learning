Pseudo Classes em CSS

Comandos Adicionais
  • display: none; faz com que o elemento não apareça ao usuário
  • display: block; faz com que o elemento apareça novamente se não estiver apareçendo
  • Você pode selecionar o filho especifico de um elemento usando > por exemplo div > p vai selecionar todos os <p> que estiverem dentro de uma <div>
Uma pseudo-classe CSS é uma palavra-chave adicionada às declarações de um seletor após um sinal de dois pontos e especificam um estado especial de um elemento. Existem várias pseudo-classes para estilos, podemos citar :hover, :visited, :active, :checked, :empty e :focus.
  • :hover
  • :visited
  • :active
  • :checked
  • :empty
  • :focus
