Peso, Estilo e font

Peso

O peso serve para deixar a letra mais grossa ou mais fina. Ele é usado com o comando font-weight: ; você pode usar valores numericos ou em inglês

Estilo

Esse comando serve para deixar a fonte em italico, você pode faze-lo com font-style: ; pode receber o valor italic

font

O font: ; é um comando para modificar a sua fonte em geral, você deve usar o comando e colocar os valores dos comandos na seguinte ordem:

  • font-style: ;
  • font-weight: ;
  • font-size: ;
  • font-family: ;
Exemplo:
font: italic bold 1.5em Verdana, sans-serif
