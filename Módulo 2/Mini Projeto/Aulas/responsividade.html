Sites responsivos com a HTML e com as CSS

O que é responsividade?

Responsividade é o seu site se adaptar a diferentes tamanhos de tela, como usar seru site em celulares, computadores e até mesmo TVs

Texto responsivo

Para tornar um texto responsivo, você seleciona nas CSS a caixa em que o texto está, e coloca um tamanho minimo e um tamanho máximo, é recomendável que você não defina um tamanho miníno para que mesmo nas menores telas seu site funcione, porém para telas maiores é bom colocar um tamanho máximo que fique confortável de ler, para que o usuário não tenha que ficar atravessando toda a tela com os olhos, recomendo que coloque o tamanho maxímo entre 600px a 800px, exemplo do código:

main{min-width: none;
max-width: 800px;}

Imagens responsivas

Para criar imagens responsivas ou vídeos responsivos, você deve criar um bloco para eles, colocá-los no centro usar width: 100%; dessa forma ele vai sempre ocupar o tamanho completo da caixa em que se localiza, você também pode definir se uma imagen vai ficar pequena, já que algumas imagens no nosso site nós queremos que independente do viewport ela seja pequena: width: 30%;, enfim, isso você vai configurar com suas classes, exemplo de código:

img, iframe, video{display: block; margin: auto; width: 100%;}
img.pq, iframe.pq, video.pq{width: 30%}

Lembrando que <iframe> é a tag que é utilizada para incorporar vídeos no youtube, você pode colocar classe e definir estilos para ela sem problemas.
