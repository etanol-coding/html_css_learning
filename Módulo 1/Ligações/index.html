Ligações em html

O que são as ligações?

O link ou ligação é a essência fundamental do hipertexto. Ela nos permite ligar documentos entre si e permitir a navegação entre essas páginas. Nesse capítulo vamos aprender a configurar as âncoras para vários comportamentos, desde o mais simples que é criar um link interno até ligações que habilitam downloads de arquivos.

Como fazer uma âncora?

Para fazer uma âncora é só usar a tag <a> e usar o parâmetro href="" e colocar o link que for dentro das aspas.

site legal

O parâmetro rel=""

Esse parâmetro rel serve para mostra basicamente que link é esse para o google.

rel="next" indica que o link é para a próxima parte do documento atual
rel="prev" indica que o link é para a parte anterior do documento atual
rel="author" indica que é um link para o site do autor do artigo atual
rel="external" indica que é um link para outro site que não faz parte do site

Quando usar rel="nofollow"?

Se uma informação não for tão relevante assim, mas você sentir que precisa de um link, optar pelo parâmetro rel="nofollow" é melhor. Certifique-se também que links de anúncio, afiliados, e afins, não sejam rel="dofollow". Use o parâmetro rel="nofollow" toda vez que inserir links que você não deseja que sejam seguidos de volta. Isso é fundamental quando for preciso citar links de concorrentes, não apenas em termos de nicho, mas também de palavras-chave.

E quando usar rel="dofollow"?

Antes de usar rel="dofollow" nos seus links, certifique-se de que eles são relevantes para a sua postagem e que não estão ali somente por conta da linkagem. Do contrário, os motores de busca podem considerar o seu post como spam

O parâmetro hreflang=""

Com esse parâmetro você pode definir qual é a ligua do link, por exemplo "pt-br" (português brasileiro) ou "en" (inglês) para definir a língua do site, para que também os tradutores automaticos funcionem.

O parâmetro target=""

Esse parâmetro permite que você defina como você quer que o link seja aberto

target="_self"

O target="_self" abre o link na mesma aba do seu site, recomendado para uma continuação do seu site, ou outra parte do seu site.

Você pode colocar downloads, faça o download desse arquivo, para colocar downloads use o parâmetro download="" com o nome do arquivo e a tag type="" que você pode usar diferentes formas para diferentes formas de arquivo:
