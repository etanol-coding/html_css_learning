CSS inline

As CSS inline são usadas na própria tag, você aplica o estilo somente nessa tag.

Para usar os comandos dessa forma: Na tag desejada use o parâmetro style="" e dentro das aspas coloque os comandos.

Essa forma de usar os comandos nas CSS é pouco recomendada, apenas use ela em casos bem específicos e pontuais, você deve evita-la, mais lembre-se não é errado usar ela, a questão é que você só deve usar dessa forma bem poucas vezes no seu site, por exemplo quando você vai aplicar esse estilo literalmente somente nessa tag, apesar de que mesmo assim é bom evitar.
