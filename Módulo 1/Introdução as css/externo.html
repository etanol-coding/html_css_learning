CSS externo

As CSS externas são usadas em um arquivo diferente, você pode colocar os comandos dentro do arquivo e usar para diversas páginas do seu site ao mesmo tempo

Para usar as CSS externas, dentro de head você digita link:css e aperta enter. Dentro do parâmetro href="" você coloca o nome do seu arquivo,

lembrando que o arquivo obrigatóriamente tem que ter a extensão .css

Essa é a forma de usar as CSS mais recomendável, pois um único arquivo pode já ser usado para diversas páginas, o que economiza tempo, código, e deixa tudo mais prático e otimizado. Sempre que puder use as CSS dessa forma.
