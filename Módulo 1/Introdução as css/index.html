Introdução as CSS

Nesse capítulo iremos aprender o básico das css e aprenderemos dois comandos super basicos das CSS também.

Vamos primordialmente entender como e onde usar as css. são três formas de usar comandos nas css:

inline
interno
externo

Antes de tudo você precisa entender que todo comando nas CSS tem seu:

Declaração(declaration):
Basicamente cada comando nas CSS é uma declaração
Seletor (selector):
Seleciona qual elemento sera estilizado
Propriedade (property):
A propriedade determina o que você quer estilizar nesse elemento, a cor? a fonte? e ETC
Valor da propriedade (property value):
Seleciona de que modo você quer mudar a propriedade, qual cor você vai usar? qual fonte você vai usar? e ETC
