Semântica no HTML5

A grande diferença do html4 para o html5 foi a semântica, agora são utilizadas tags semânticas

O que é a tal "Semântica"?

Semântica é basicamente sobre dar sentido nas tags. Antes as tags não tinham nomes com muito sentido, era basicamente as css na html, então para acabar com essa bagunça agora as tags não servem mais para estilo.

Mais então pra que serve as tags da html?

Então, na verdade as tags até mudam o estilo (Apesar disso, NUNCA use as tags html para mudar efetivamente o estilo, faça isso nas css). Mais na verdade as tags em html servem para você deixar o seu site mais dinâmico, deixar que o google entenda o que é cada texto no seu site e ETC.

Agora bora aprender algumas tags de texto no html5!

Tags de Endereço

A tag <address> serve para localizar um endereço. Isso serve para mostra para o google que esse texto é um endereço, o google pode até te mostrar um caminho para o endereço! Essa tag também coloca o texto em itálico, mais novamente não deve ser usada para isso, isso deve ser feito no css.
